# Welcome

Welcome to the Do11y integration test site built with Hugo Docsy.

## Getting started

Follow these steps:

```bash
pip install example-sdk
```

<button class="btn" aria-label="Copy to clipboard">Copy</button>

Then configure your application:

```python
from example_sdk import init

init(
    api_key="your-api-key",
    environment="production",
)
```

## FAQ

<details>
<summary>How do I install the SDK?</summary>

Run `pip install example-sdk` in your project directory.
</details>

<details>
<summary>What environments are supported?</summary>

The SDK supports Python 3.9+ on Linux, macOS, and Windows.
</details>
